# Sistema de Citas

## Descripción
Este es un programa creado escolarmente con el propósito de ser de uso libre. Es portable, lo que significa que puede ser ejecutado en diferentes plataformas sin necesidad de instalación.

## Contenido

- [Uso del Programa](#uso-del-programa)
- [Créditos](#créditos)
- [Licencia](#licencia)

## Instalación y Configuración
El programa no requiere instalación. Simplemente descarga el archivo ejecutable desde la sección de lanzamientos y ejecútalo en tu sistema operativo. Asegúrate de tener los permisos necesarios para ejecutar el programa si estás en un entorno controlado.

## Uso del Programa
El programa es fácil de usar y no requiere configuración adicional. Abre el archivo ejecutable y sigue las instrucciones en pantalla para utilizarlo.
